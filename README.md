**GooseOS** is a purely hobbyist operating system created for hands-on learning in low-level systems and kernel development.  
The project focuses on understanding how an OS works from the ground up, including bootloading, memory management, and hardware interaction.

---

### Project Goals
- Learn x86 kernel development from scratch
- Gain deeper experience with C and Assembly
- Understand bootloaders, protected mode, and CPU initialization
- Experiment with low-level hardware access

---

### Technical Details
- **Architecture:** x86 (32-bit)
- **Type:** Monolithic kernel (educational)
- **Status:** Early development / experimental
- **Purpose:** Learning & exploration (not production-ready)
